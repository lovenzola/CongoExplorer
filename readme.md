# CongoExplorer

## Description du projet

**CongoExplorer** est une application web interactive qui permet de découvrir la richesse naturelle de la République démocratique du Congo à travers une expérience immersive et éducative.

Le site propose une exploration de la **faune congolaise**, accompagnée d’un **quiz interactif** permettant de tester les connaissances de l’utilisateur après la découverte.

---

##  Objectifs

* Mettre en valeur la biodiversité congolaise
* Offrir une expérience utilisateur fluide et immersive
* Combiner **design moderne** et **logique interactive (JavaScript)**
* Initier à une approche pédagogique via un quiz

---

##  Fonctionnalités principales

### Exploration

* Présentation de la faune congolaise (léopard, okapi, gorille, crocodile)
* Affichage dynamique des descriptions
* Interface responsive adaptée à différents écrans

###  Quiz interactif

* Questions à choix unique (radio)
* Navigation entre questions (suivant / précédent)
* Système de score
* Vérification des réponses

###  Compte à rebours

* Timer de 5 minutes pour compléter le quiz
* Fin automatique lorsque le temps est écoulé

### Interface utilisateur

* Menu burger interactif
* Animations CSS
* Design inspiré des documentaires (style immersif)

---

##  Technologies utilisées
